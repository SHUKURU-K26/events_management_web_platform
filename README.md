# SMART EVENT RESOURCE MANAGEMENT PLATFORM

A modern web Platform built for an Event Handling Company for  managing
events, staff assignments, resource allocation, expenses, client bookings, and Monthly Report
Generation of Transactions

## Main Objectives
● Clients can request/book events
● Admins can manage operations
● Team members can track assigned tasks
● Resources and expenses can be monitored
● Reports and analytics can be generated

## Commands
//Prepare the Client
1. npm install pnpm //for more disk-space efficiency, significantly faster at installing packages,
2. pnpm create vite@latest
     -folder-name: client
     -react
     -javascript
3. cd client
4. pnpm install // To install all packages found in node modules

//Prepare the Server
1. root>mkdir server
2. cd server
3. pnpm init // To initialize the package 
4. pnpm install // All packages for powering server
